# Book covers and credits

This folder stages book-cover images and credit notes for the OFBizian books
page: <https://www.ofbizian.com/p/books.html>.

## Files

- `covers-web/` - publishable upright PNG covers, resized to 1200 x 1600.
- `contact-sheet-upright.png` - visual index for checking the cover filenames.

The current covers are PNGs because they are simple to publish from GitHub. For
the Blogger page, JPG or WebP copies would load faster if page weight becomes a
problem.
